# UI Components

Skills for frontend development, UI libraries, design systems, and component frameworks.

## Available Skills

### shadcn-ui

Modern UI component library integration with Tailwind CSS.

**Capabilities:**
- Component installation and configuration
- Form building with React Hook Form and Zod validation
- Data tables with sorting, filtering, and pagination
- Complex UI patterns (dialogs, dropdowns, sheets)
- Tailwind CSS integration
- Accessibility-first components

**Use Cases:**
- Building forms with validation
- Creating data-heavy interfaces
- Implementing design system components
- Rapid prototyping with pre-built components

**Installation:**
```bash
# Upload shadcn-ui-skill.skill to Claude Code via Settings → Skills
```

**Usage Examples:**
- "Create a user registration form with validation"
- "Build a data table with sorting and filtering"
- "Add a command palette to the app"
- "Implement a multi-step form wizard"

## Component Categories

The shadcn-ui skill includes knowledge of:
- **Forms** - Input, Select, Textarea, Checkbox, Radio, Switch
- **Data Display** - Table, Card, Badge, Avatar, Calendar
- **Feedback** - Alert, Toast, Dialog, Sheet, Popover
- **Navigation** - Tabs, Breadcrumb, Pagination, Command
- **Overlays** - Modal, Drawer, Tooltip, Dropdown Menu
