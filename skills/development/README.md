# Development Tools

Skills for software development workflows, version control, CI/CD, and code quality.

## Available Skills

### github-workflow

GitHub workflow automation and management skill.

**Capabilities:**
- Create and manage GitHub Actions workflows
- Handle pull requests with proper formatting
- Manage issues and project boards
- Follow conventional commit standards
- Generate workflow templates

**Use Cases:**
- Automating CI/CD pipelines
- Setting up automated testing
- Creating release workflows
- Managing repository automation

**Installation:**
```bash
# Upload github-workflow.skill to Claude Code via Settings → Skills
```

**Usage Examples:**
- "Create a CI workflow for a Node.js project"
- "Set up automated testing with GitHub Actions"
- "Generate a release workflow with semantic versioning"
