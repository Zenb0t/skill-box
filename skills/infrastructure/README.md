# Infrastructure

Skills for backend services, databases, cloud platforms, and DevOps tools.

## Available Skills

### supabase-cli

Supabase database and backend management skill.

**Capabilities:**
- Database schema design and migrations
- Authentication and authorization setup
- Real-time subscription configuration
- Edge functions development
- Storage bucket management
- Row-level security (RLS) policies

**Use Cases:**
- Setting up a new Supabase project
- Creating database migrations
- Implementing authentication flows
- Building real-time features
- Managing database security

**Installation:**
```bash
# Upload supabase-cli.skill to Claude Code via Settings → Skills
```

**Usage Examples:**
- "Create a Supabase migration for a users table"
- "Set up authentication with social providers"
- "Create an edge function for email notifications"
- "Implement RLS policies for multi-tenant app"
